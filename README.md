# Self by Saint - 216v

**Selfbot Discord CLI** - Registro via Token e Controle via terminal.

Um .bat que conecta à sua conta Discord usando WebSockets puros e permite automação da sua conta.

## 🚀 Funciolidades

- ✅ **Clear DMs** - Limpeza inteligente de mensagens diretas com backup opcional
- ✅ **FarmCall** - Entre em voice channels para farmar (múltiplos farms simultâneos)
- ✅ **Clear Servers** - Sair de servidores de forma organizada.
- ✅ **Clear Friends** - Limpe sua lista de amigos completamente.
- ✅ **Exportação de Conversas** - Faça backup das suas DMs.


## 📋 Requisitos (Já baixados junto do repositório)

- Node.js 18+ (versão com suporte a ES modules)
- npm 9+

## 🔧 Utilização

```bash
# 1. Baixe o repositório em .zip e extraia.

# 2. Entre na pasta

# 3. Execute o arquivo .bat chamado "self-saint.bat"

```

### Primeiro Login

Na primeira execução, você será solicitado a:
1. Registrar uma conta com seu **Token**;
2. Depois disso, basta utilizar. As contas ficam registradas via banco local dentro dos arquivos baixados.
